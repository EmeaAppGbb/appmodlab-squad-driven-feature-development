ARCHITECTURE:
- Polyglot microservice architecture
- api/ - TypeScript Fastify server (port 3000)

  - Plugin system for database and Redis connections
  - JWT authentication support
  - JSON Schema validation
- ml-service/ - Python FastAPI server (port 8000)
  - Recommendation engine (stub)
  - PostgreSQL direct connection
- database/ - Schema and seed data
  - 5 tables: users, products, orders, order_items, reviews
  - 20 users, 50 products, 100 orders seeded
