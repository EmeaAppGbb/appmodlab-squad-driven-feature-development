WHAT COULD IMPROVE 🔧

1. Integration Tests
   Unit tests are strong but integration tests with real PostgreSQL
   would catch query-level bugs. Recommend Docker-based test suite.

2. Redis Caching
   Product rating aggregates are computed on every request.
   Adding Redis cache with TTL would improve performance.

3. Authentication
   Review create/update/delete should require JWT authentication.
   Currently any user_id can be passed in the request body.

4. Rate Limiting
   No rate limiting on review creation. Could be abused.

5. asyncpg Dependency
   ML service uses asyncpg but pyproject.toml lists psycopg2-style deps.
   Should add asyncpg to project dependencies.
